📖 ดูคู่มือตั้งค่า Google Sheets
— ยังไม่ได้เชื่อมต่อ
📋 คู่มือตั้งค่า Google Sheets (ทำครั้งเดียว)
ขั้นที่ 1: สร้าง Google Sheets ใหม่ แล้วตั้งชื่อ Sheet ว่า Products
ขั้นที่ 2: ไปที่เมนู Extensions → Apps Script
ขั้นที่ 3: ลบโค้ดเดิมทิ้ง แล้ววางโค้ดด้านล่างแทน:
function doGet(e) { var sheet = SpreadsheetApp.getActiveSheet(); var data = sheet.getDataRange().getValues(); var headers = data[0]; var rows = data.slice(1).map(function(row) { var obj = {}; headers.forEach(function(h, i) { obj[h] = row[i]; }); return obj; }); return ContentService .createTextOutput(JSON.stringify({status:'ok',data:rows})) .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { var sheet = SpreadsheetApp.getActiveSheet(); var payload = JSON.parse(e.postData.contents); sheet.clearContents(); if(payload.data && payload.data.length > 0) { var headers = Object.keys(payload.data[0]); sheet.appendRow(headers); payload.data.forEach(function(row) { sheet.appendRow(headers.map(function(h){ return row[h]||''; })); }); } return ContentService .createTextOutput(JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); }
ขั้นที่ 4: กด Deploy → New deployment
ขั้นที่ 5: เลือก Type: Web app → Execute as: Me → Who has access: Anyone → กด Deploy
ขั้นที่ 6: Copy URL ที่ได้ วางในช่องด้านบน
✅ หลังตั้งค่าแล้ว กด "ส่งขึ้น Sheets" ครั้งแรกเพื่ออัปโหลดข้อมูลปัจจุบัน จากนั้นทุกอุปกรณ์กด "ดึงจาก Sheets" เพื่อรับข้อมูลล่าสุดได้เลย
📱 ติดตั้งบนมือถือ / Tablet

แอปนี้เป็น PWA (Progressive Web App) ติดตั้งได้โดยไม่ต้องผ่าน App Store

🍎 iPhone / iPad
1
เปิดใน Safari
ต้องใช้ Safari เท่านั้น (Chrome บน iOS ไม่รองรับ)
2
กดปุ่ม Share (กล่องลูกศรขึ้น)
อยู่ที่ด้านล่างของหน้าจอ iPhone หรือด้านบนบน iPad
3
เลือก "Add to Home Screen"
เลื่อนลงในเมนู แล้วกด "Add to Home Screen" หรือ "เพิ่มในหน้าจอหลัก"
4
กด Add / เพิ่ม
แอปจะปรากฏบน Home Screen เหมือน App ทั่วไป
🤖 Android
1
เปิดใน Chrome
ใช้ Google Chrome บน Android
2
กด ⋮ เมนู (3 จุด)
มุมขวาบนของเบราว์เซอร์
3
เลือก "Add to Home screen"
หรืออาจมี Popup ถามอัตโนมัติ "Install app?"
💡 วิธี Host แอปนี้ฟรีบน GitHub Pages

1. สมัคร GitHub ที่ github.com (ฟรี)
2. สร้าง Repository ใหม่ชื่ออะไรก็ได้
3. อัปโหลดไฟล์ index.html ขึ้นไป
4. ไปที่ Settings → Pages → เปิด GitHub Pages
5. ได้ URL เช่น yourname.github.io/nutrition
6. แชร์ URL นี้ให้ทีมใช้งานได้เลย

อัปเดตข้อมูล: แก้ไขไฟล์ใน GitHub → ทุกอุปกรณ์อัปเดตอัตโนมัติ
